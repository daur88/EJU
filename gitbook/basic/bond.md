
## 金属結晶:

```yaml
性质:
  - 导电
  - 常温固体
  - 展性
  - 延性
包括:
  - 金属 del Hg

```

## イオン結晶:

```yaml
性质:
  - 常温固体
  - 熔点高
  - 沸点高
  - 不导电 if solid
  - 导电 if aq

```

## 共有結合:

```yaml
性质:
  - 常温是固体
  - 熔点高
  - 沸点高
  - 坚硬
  - 不导电 del 黒鉛
```

## 水素結合:

```yaml
定义: 相互吸引力很大的分子间力
```
