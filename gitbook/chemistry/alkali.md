## 酸·塩基の性質:

```yaml
酸:
  性质:
    - 和金属的反应生成H2
    - 使青色リトマス紙变赤
    - 使BTB(ブロモチモールブルー)变黄
    - 与塩基反应抵消塩基性
塩基:
  性质:
    - ぬるぬる
    - 使赤色リトマス紙变青
    - 使BTB变青
    - 与酸反应抵消酸性
  包括:
    - アルカリ金属
    - アルカリ土類金属

笔记: 溶于水的塩基叫アルカリ 水溶液的性质叫做アルカリ性

```

## 酸·塩基定義:

```yaml
アレーニウス:
  定义: |
        酸  : 溶于水生成 H+(H3O+)的物质
        塩基 : 溶于水生成 O- 的物质

ブレンステッド·ローリー:
  定义: |
        酸  : 给予对方 H+ 的物质
        塩基 : 得到 H+ 的物质
  笔记: 广义
```
