## 電離度:

```yaml
计算公式: α = 电解的总物质的量/溶解的总物质的量  (0<α≦1)

```

## 中和反応:

```yaml
定义: 酸 + 塩基 → 塩 + 水
计算公式: a·c·v = b·c'·v'

```

## 塩の分類:

```yaml
正塩: 没H、OH剩下的盐
酸性塩: 剩下H的盐
塩基性塩: 剩下OH的盐

```

## 酸化物の分類:

```yaml
酸性酸化物:
  定义: 非金属酸化物
  包括:
    - 非金属 + O
塩基性酸化物:
  定义: 金属酸化物
  包括:
    - 金属 + O
両性酸化物:
  定义: 两性元素酸化物
  包括:
    - 両性金属 + O
```
