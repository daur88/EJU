## Fe:

```yaml
赤鉄鉱:
  化学式: Fe2O3
  color: 赤褐色

黄鉄鉱:
  化学式: FeS
銑鉄:
  包括: C
  定义: 4% C
  性质: もろい
鋼:
  包括: C
  定义: 0.02~2
ステンレス鋼:
  包括:
    - Fe
    - Cr
    - Ni
  用途: 精密仪器
沈殿:
  NaOH:
    - Fe2+ => 綠白色
    - Fe3+ => 赤褐色
  Fe(CN):
    - Fe2+ + Fe2+ => 青白色
    - Fe2+ + Fe3+ => 濃青色
    - Fe3+ + Fe3+ => 褐色
    - Fe2+ + Fe2+ => 青白色
  KSCN:
    - Fe2+ => 変化なし
    - Fe3+ => 血赤色溶液

```
