# その他

## 綿、麻:

```yaml
燃え方: 燃え易くパッと燃え、炎を離して燃える。
匂い: 紙を焼く匂い
灰の状態: 灰色の軟らかい灰を少し残す。

```

## 絹:

```yaml
燃え方: ジリジリと縮れながら早く燃える。
匂い: 毛を焼く匂い
灰の状態: 黒褐色の塊になるが押すと容易に潰れる。

```
