# 酵素·核酸

## 酵素:

```yaml
component: mainly タンパク質
特異性:
  定义: 只作用于特定物质
最適PH: 最合适反应的PH
最適温度: 最合适反应的温度
補酵素: 反应必须的辅助因子

```

## 核酸:

```yaml
定义: 含有生物遗传情报
结构:
  - 多数のヌクレオチド ->脱水缩合
  - 鎖状

```

## ヌクレオチド:

```yaml
components:
  - 糖
  - リン酸
  - 塩基

```

## リボース:

```yaml
定义: 核糖

```

## DNA:

```yaml
定义: 双螺旋发现者 ワトソン&クリック
别名: デオキシリボ核酸
结构:
  - 中间含有核
  - 2本鎖の構造を持つ
```

## RNA:

```yaml
别名: リボ核酸
定义: 核和细胞质中両方存在
结构:
  - 1本鎖の構造を持つ
用途: 参加蛋白质合成

```

## リボソーム:

```yaml
定义: 核糖体
```
